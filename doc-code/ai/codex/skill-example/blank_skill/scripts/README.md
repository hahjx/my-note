# scripts 目录说明

`scripts/` 用于保存 Skill 可重复使用的脚本。

脚本适合处理稳定、重复、容易写错的操作。这样 Codex 不需要每次临时重写一段命令或程序。

## 适合放在这里的内容

- 固定格式转换脚本。
- 校验脚本。
- 批处理脚本。
- 生成模板文件的脚本。
- 从结构化文件提取信息的脚本。

## 不适合放在这里的内容

- 只运行一次的临时代码。
- 需要频繁人工改动的脚本。
- 没有被 Skill 文档说明用途的脚本。
- 会默认执行危险操作的脚本。

## 示例

```text
scripts/
  validate-input.js
  generate-report.py
  normalize-data.mjs
```

在 `SKILL.md` 中应该说明脚本何时运行、输入是什么、输出是什么。
